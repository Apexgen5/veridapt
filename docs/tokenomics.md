# Veridapt Tokenomics (VERI)

## Token Overview

**VERI** is the native utility and governance token of the Veridapt network.

### Token Utility

#### 1. Protocol Fees
- Users pay in VERI (or stablecoins via AMM) for:
  - Zero-knowledge proof verification
  - Credential issuance and renewal
  - Oracle data access
  - Cross-chain attestations

#### 2. Staking & Security
- Network validators stake VERI to participate in consensus
- Staking rewards from protocol fees
- Slashing mechanisms for malicious behavior

#### 3. Governance
- VERI holders vote on:
  - Protocol parameters and upgrades
  - Predicate whitelisting
  - Treasury allocation
  - Risk parameter adjustments

#### 4. Enterprise Subscriptions
- Premium features unlocked with VERI lock-ups
- Discounted rates for stakers
- Priority access to new features

### Revenue Streams

| Source | Distribution |
|--------|---|
| Attestation Fees | 50% to validators, 50% to treasury |
| Enterprise Subscriptions | 60% to treasury, 40% to validator fund |
| Premium Services | 70% to treasury, 30% to community |

### Emission Schedule

**Phase 1 (Year 1):** Base emissions for incentivizing early adoption  
**Phase 2 (Years 2-3):** Decreasing emissions as protocol matures
